## Perform basic steps
```
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install gfortran
```

## Install Pip
```
wget https://bootstrap.pypa.io/get-pip.py
sudo python3 get-pip.py
```
## Download python packages for future

```
cd ~/Downloads
mkdir python_packages && cd python_packages
pip3 download numpy 
```
